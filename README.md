# MIPS16

Izvedba enostavnega enocikelnega procesorja MIPS, ki predstavlja prilagoditev (bolj znane) 32-bitne arhitekture na 16-bitno. Pri tem je 16-bitni podatkovni pomnilnik loÄen od 8-bitnega ukaznega pomnilnika (Harvardov model). Za razlago procesorja glej prosojnice predavanj predmeta Digitalna vezja.
